== Description ==
Delete your any blog post, page, or any other custom post type automatically after a certain time. This plugin will help you to delete a post after expiration of the given time which you can set during a post publish. Set a time from the time and date input field and be tension free! The plugin will do the rest for you. You can also select from the options checkbox in the Auto Delete Posts menu page where to add this automatic post deletion functionality.


== Installation ==

**The easy way.**

Go to your WordPress Dashboard. Navigate to Plugins > Add New and then search for "Auto Delete Post". Click on Install and then Activate the Plugin.

That's it, Auto Delete Post Plugin is now activated on your site!

**The hard way..**

Download "Auto Delete Post WordPress Plugin"  and then extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
